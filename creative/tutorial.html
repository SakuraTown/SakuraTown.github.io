# 创造服新手指引

樱花镇创造服是一个意在激发玩家玩家创造力的服务器，目前的主要内容是建筑和红石，如果有更多需要可以私聊服主QingNiao或管理 EnTIv 申请哦

# 挥洒创意

当您进入到创造服时，您可以输入 /p auto 来领取一块地皮，在该地皮上尽情挥洒您的创意。有关地皮的更多信息，请查阅 地皮教程 来了解更多

初次进入服务器您会处于 萌新 级别，只有最基础的游戏权限，您需要通过审核才能提升自己的等级。目前创造服主要审核建筑作品，详细审核内容请看 审核机制 来了解更多

# 审核机制

建筑审核由高级建筑师以上的等级进行审核，且不接受像素画以及像素人等过于直白的建筑，如果建筑带有环境，不占用主体所需面积要求。接下来您会看到有关服务器各个等级的审核要求

# 初级建筑师

主体建筑大小最小 25x25，中式民居类和现代别墅类要有内饰，幻想类建筑要整体完整，有规划和细节，其他样式要具备明显的风格要素。

# 中级建筑师

主体建筑大小最小 50x50，与一审的建筑要求一致，整体建筑外观和区域划分要合理

# 高级建筑师

主体建筑大小最小 150x150，与一审的建筑要求一致，建筑规模应该是一个完整体，类似小项目、建筑群等

# 创世神

主体建筑大小最小 250x250，与一审的建筑要求一致，建对一种掌握的风格要有自己独特的领悟和见解，创造自己的风格类型，不违背大众审美观
