# 樱花镇规则

樱花镇使用严格的规则来约束玩家，在樱花镇内请务必遵守以下规则 否则将会受到严厉的处罚，如果玩家产生纠纷，如无法和解将会按照规则严肃处理

如多次犯错屡教不改或开小号试图逃避惩罚 将会进行从重处理，甚至将会翻倍封禁天数，请不要挑战规则

犯错后认错态度良好有可能能够减轻处罚 还请各位安安分分的再游戏里游玩，尽量不要和他人产生冲突哟

如果他人开挂请不要公屏举报，否则将以污蔑他人罪论处 请私聊 EnTIv 附上视频证据进行举报，等待管理仔细查明后进行处理

# 禁止损害他人财物

判断依据: 在未经他人同意的情况下，拿走或损坏他人的物品，农作物，动物，装备和财产等

处理措施: 犯错将会进行全账号封号，直到主动加群私聊群主认错并补偿他人损失后解封

# 禁止破坏他人建筑

判断依据: 对他人建筑或周边进行破坏，堵路和放岩浆等使他人的建筑遭到破坏，影响美观性或无法居住

处理措施: 犯错将会进行封号，直到主动加群私聊群主认错并补偿他人损失后解封

# 禁止语言辱骂他人

判断依据: 公然侮辱，捏造事实或暗示对他人进行辱骂诽谤等，引起他人反感，不舒适或影响服务器风气

处理措施: 犯错依据严重程度进行警告或封禁处理，最严重的行为有可能导致永久封禁

# 禁止使用辅助工具

判断依据: 在服务器内使用作弊客户端，作弊MOD等，其他辅助工具如不确定是否可以使用请咨询服主

处理措施: 犯错依据严重程度进行警告或封禁处理，最严重的行为有可能导致永久封禁

# 禁止传播低俗文化

判断依据: 物品，宠物，小黄书和用户名使用低俗语言，影响服务器风貌，对他人进行侮辱，传播色情暴力等

处理措施: 首次犯错警告一次处理，后续将会依据严重程度封禁 1~7 天处理

# 禁止利用游戏漏洞

判断依据: 利用游戏漏洞影响游戏公平性，导致刷钱或服务器无法正常运作等问题

处理措施: 首次犯错警告一次处理，刷钱，刷物品的行为极有可能导致永久封禁!
