# 服务器介绍

樱花镇目前是一个简简单单的小服务器，基本上由服主 EnTIv 一个人慢慢肝的，虽然目前只有生存服和创造服，但是也会慢慢添加新玩法新特性哒~

服务器也不会进行删档换周目做快餐服，会花很多时间进行服务器的日常管理和新玩法制作等等，所以我相信以后一定会变成大服务器的! 到时候你们就都是老玩家啦~

服务器的游戏氛围也很不错噢，不知不觉就聚集了好多好多人呢，有许多肝帝，红石和建筑 dalao..，玩家整体素质是十分高的哦，总之~ 赶快 加入樱花镇!

# 生存服

生存服是一个比较基础的生存服，新人入服即赠送一个专属地皮，不用害怕别人破坏你的家，同时还有商店，领地和技能等等新奇有趣的玩法

生存服拥有许多有趣的玩法的同时允许建造各种红石机械与刷怪塔，尽量不破坏原版玩家的生存体验~

里面还有许许多多 dalao 玩家们自己建设的红石装置，如推币机，射箭场，竞技场和赌场等等~

还有许许多多的功能和特性尚未介绍，但是您可以查看 生存服新手指引 了解更多~

# 创造服

樱花镇创造服是由 青鸟 进行管理，樱花镇服主 EnTIv 负责幕后支持共同建设的创造服，同时由主播 萬大爷(21:00-3:00 开播) 对创造服的玩家建筑水平进行审核

创造服还有主打的 天际线项目，目标是在 10000x10000 的坐标上建满任意风格的建筑，只要您通过审核达到中级建造师等级，便可一起参与到天际线项目中~

想要更多了解创造服嘛? 查看 创造服新手指引 了解更多~
