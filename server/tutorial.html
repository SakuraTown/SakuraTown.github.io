# 加入樱花镇

您可以直接下载我们服务器的客户端来快速加入樱花镇服务器 下载完毕之后点击多人游戏即可直接加入樱花镇服务器: 点击下载

当然，如果您不想下载也没关系，樱花镇服务器版本为1.16.1群组服 您只需要使用1.16.1版本的客户端在多人游戏中添加我们的IP地址即可: sakuratown.cn

最后，请一定要加入我们的QQ群来和我们一起玩耍哟 在QQ群里您能找到朋友开黑，调戏腐竹，并且第一时间收到樱花镇的各种通知: 点击加群

# 樱花镇注册教程

如果您是第一次接触 Minecraft 服务器，完全不知道如何做也没关系，可先查看以下教程，跟着教程一步一步往下做，如如果有什么不明白的地方可以加群询问管理员

首先需要下载樱花镇客户端来加入服务器: 点击下载 下载完毕之后双击解压，耐心等待解压完毕后打开樱花镇客户端文件夹，双击打开 HelloMinecraft 启动器

如果提示需要安装 Java，那么就需要先安装 Java，而 Java 又分为 32 位和 64 位两种版本 如何知道自己的电脑适合什么版本的 Java 呢?，点击文字查看教程: 如何辨别系统是 32 位还是 64 位

Java 下载地址:
32 位 Java: 点击下载
64 位 Java: 点击下载

Java 下载完毕之后双击进行安装即可，HelloMinecraft 启动器启动之后应该看起来像这个样子

左上角拥有四个选项:

  • 配置: 保持不变即可
  • 版本: 如果无法启动请尝试切换为 1.12.2 版本
  • 登录: 如果拥有正版账号可以切换为正版账号登录，否则离线登录即可
  • 名字: 仅允许英文数字和下划线组合，不允许中文以及特殊符号

设置名字好即可点击右下角的启动 Minecraft 启动游戏
进入到主页面点击多人游戏，双击加入樱花镇服务器之后会需要您进行注册

输入如图所示指令即可成功注册，请牢记您的账号(启动器启动时设置的名字)和密码
并且不要将密码告诉他人哦，账号和密码是您登录樱花镇的唯一凭证哦，加入樱花镇教程到此完毕~

接下来选择一个您想游玩的服务器进行游玩哦
