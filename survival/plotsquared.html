# 地皮教程

每一位玩家都可以拥有一块自己的地皮作为居住区，您可以输入 /p menu 来打开地皮菜单

想要领取一个地皮是非常简单的，您可以使用 /p auto 来自动领取一个地皮，您也可以使用 /p claim 来领取脚下的地皮

# 权限设置

有了地皮当然需要邀请小伙伴一起在您的地皮上建筑，地皮提供了三种权限来管理您的地皮

  1. /p add 玩家 您在线的时候允许该玩家在您的地皮上建筑
  2. /p trust 玩家 任何时候都允许该玩家在您的地皮上建筑
  3. /p deny 玩家 禁止该玩家进入您的地皮

提示

您可以使用符号 * 来表示所有玩家，例如使用 /p deny * 拉黑所有人等等 您也可以使用 /p remove 玩家 来将某玩家恢复到默认权限

# 搬迁地皮

搬迁须知

不能搬迁已经合并的地皮 箱子商店无法正常搬迁 潜影盒搬迁有可能失败

首先先找到一个无人认领的地皮，站在无人认领的地皮上输入 /p info 可以查看到地皮的 ID，如 5;-12 就是一个地皮的 ID，然后站在自己的地皮上输入 /p move 5;-12 即可成功移动自己的地皮

# 合并地皮

当你拥有多块地皮时，只需要使用 /p claim 指令将地皮认领在一起，即可自动合并成为一个大地皮

危险

地皮合并之后，使用 /p clear 或 /p delete 指令将会清理或删除所有地皮，请先使用 /p unlink 指令取消合并再进行清理和删除地皮操作！取消合并会恢复原有地皮道路, 原有道路中央建筑会消失!

如果您的地皮使用了 /p unlink 指令来取消合并之后想要重新合并，就要站在您的地皮上，看着你想合并的地皮的方向输入 /p merge 来重新合并您的地皮

# 指令大全

/p trust 玩家 允许某玩家在您的岛屿建筑
/p untrust 玩家 禁止某玩家在您的岛屿建筑
/p h 回到自己的岛屿
/p v 玩家 拜访某玩家的岛屿
/p setalias 设置岛屿名称
/p sethome 设置岛屿传送点
/p info 查看您所在的岛屿信息
/p clear 清空你的岛屿(会删除建筑)
/p delete 删除您的岛屿(会删除建筑)
/p list 玩家 列出该玩家的所有岛屿
/p setdescription 设置岛屿描述
/p middle 将你传送到岛屿的中心点
/p setowner 玩家 将岛屿给予另一名玩家
/p kick 玩家 将某玩家踢出你的岛屿
/p rate 分值 给一个岛屿评分
