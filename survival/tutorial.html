# 生存服新手指引

樱花镇生存服是一个以原版生存为主，搭配多种扩展玩法的生存服。如果你是萌新请务看完新手指引，一般您遇到的问题在官网都会有详细的解答噢~ 遇到问题再也不用求人啦

# 开始生存

当你进入樱花镇生存服时，您所在的世界为樱花镇的主城，主城是无法进行破坏的，您需要传送到其他世界才可以开始生存

您可以看到樱花娘赠送了您一套石制工具和一个指南针，只需把指南针拿在手上然后右键或输入 /menu 指令即可打开传送菜单

然后您可以点击菜单上的木头 (南海水域) 来传送到生存服的资源世界来收集生存所必须的资源，如果您是第一次接触 Minecraft 的萌新，您还可以查看 菜鸟手册 来学习 Minecraft 的生存玩法

# 安家乐业

当您获取到一定资源时，就需要找地方安家了呢。建议新手可以输入指令 /p auto 来领取一个专属于你的地皮，地皮内他人无法破坏，您可以查看 地皮教程 来了解更多

除了地皮之外，您也可以在野外居住, 但是请不要考验其他玩家的素质，一定要好好把箱子锁上，然后使用 领地插件 进行圈地，不要给熊孩子任何可乘之机。

警告

如果没钱圈地造成的财务损失不会赔偿，对熊孩子也会从轻处理，所以请一定要圈地哦

# 常用指令

# 菜单指令

/menu 打开传送菜单
/shop 打开系统商店
/p menu 打开地皮菜单

# 传送指令

/spawn 回到主城
/warp 坐标 传送到服务器的特殊坐标 /tpa 玩家 请求传送到某玩家身边
/tpahere 玩家 传送到某玩家身边

# 经济指令

/bal 查看自己的金钱
/baltop 查看金钱排行榜 /pay 玩家 金钱 给予某玩家指定数量的金钱

# 聊天指令

/m 玩家 私聊一名玩家
/r 回复最后一名与你私聊的玩家

# 其他知识

  • 右键楼梯可以坐下
  • 服务器全服死亡不掉落
  • 用鸡蛋右键砸小动物可以扑捉
  • 聊天输入 [i] 可以展示手上物品
  • 使用告示牌右键箱子可以直接锁住箱子
